    public decide(state: object, rqid: number): Decision source

    Abstract function that must be overridden.

    Params:

    NameTypeAttributeDescription
    state object

    a State object. This object has the following properties:
    state.self Object

    Details about your pokemon.
    state.self.active PokemonData | Array<PokemonData>

    Contains any of your mons who are "active", i.e. on the front lines this turn. This is intended to be a single Pokemon in Singles matches and an array of Pokemon in Doubles and Triples.
    state.self.reserve Array<PokemonData>

    All the Pokemon on your side. Note that this includes your active Pokemon, who have the property 'active' set to true. You can switch into any Pokemon where neither 'active', 'disabled', nor 'dead' are true.
    state.opponent Object

    Details about your opponent's pokemons.
    state.opponent.active PokemonData | Array<PokemonData>

    Contains any of your opponent's mons who are "active", i.e. on the front lines this turn. This is intended to be a single Pokemon in Singles matches and an array of Pokemon in Doubles and Triples.
    state.opponent.reserve Array<PokemonData>

    All the Pokemon on your opponent's side, that you have seen.
    rqid number

    The request ID. This is needed for server replies and is not really something you need to worry about.

    Return:

    Decision

    a Decision object, or a Promise that resolves to a Decision object.
